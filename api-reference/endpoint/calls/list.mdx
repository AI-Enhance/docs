---
title: 'List Call Logs'
api: 'GET /api/call-logs'
description: 'Retrieve call logs with optional filtering'
---

<Note>
  **Core API**: Get a paginated list of call logs with filtering options.
</Note>

## Query Parameters

<ParamField query="page" type="integer">
  Page number for pagination (default: 1)
</ParamField>

<ParamField query="per_page" type="integer">
  Number of records per page (default: 10)
</ParamField>

<ParamField query="start_date" type="string">
  Filter calls after this date (ISO 8601)
</ParamField>

<ParamField query="end_date" type="string">
  Filter calls before this date (ISO 8601)
</ParamField>

<ParamField query="status" type="string">
  Filter by call status: "COMPLETED", "FAILED", "BUSY"
</ParamField>

<ParamField query="outcome" type="string">
  Filter by call outcome: "APPOINTMENT_SET", "QUALIFIED_LEAD", etc.
</ParamField>

## Response

<ResponseField name="success" type="boolean">
  Indicates if the request was successful
</ResponseField>

<ResponseField name="data" type="object">
  <Expandable title="Properties">
    <ResponseField name="calls" type="array">
      Array of call log objects
      
      <Expandable title="Call Log Object">
        <ResponseField name="id" type="string">
          Unique identifier for the call
        </ResponseField>
        <ResponseField name="agent_id" type="string">
          ID of the AI agent that handled the call
        </ResponseField>
        <ResponseField name="customer_phone" type="string">
          Customer's phone number
        </ResponseField>
        <ResponseField name="status" type="string">
          Call status: "COMPLETED", "FAILED", "BUSY"
        </ResponseField>
        <ResponseField name="duration" type="integer">
          Call duration in seconds
        </ResponseField>
        <ResponseField name="outcome" type="string">
          Call outcome: "APPOINTMENT_SET", "QUALIFIED_LEAD", etc.
        </ResponseField>
        <ResponseField name="started_at" type="string">
          Call start time (ISO 8601)
        </ResponseField>
        <ResponseField name="ended_at" type="string">
          Call end time (ISO 8601)
        </ResponseField>
      </Expandable>
    </ResponseField>
    <ResponseField name="total" type="integer">
      Total number of calls matching the filters
    </ResponseField>
    <ResponseField name="page" type="integer">
      Current page number
    </ResponseField>
    <ResponseField name="per_page" type="integer">
      Number of records per page
    </ResponseField>
  </Expandable>
</ResponseField>

## Example Request

```bash
curl --request GET \
  --url 'https://api.aienhance.net/v1/api/call-logs?page=1&per_page=10&status=COMPLETED' \
  --header 'Authorization: Bearer YOUR_ACCESS_TOKEN'
```

## Notes

- Results are sorted by start time in descending order (newest first)
- Phone numbers are returned in E.164 format
- Maximum date range for filtering is 90 days
- Time values are always in UTC
