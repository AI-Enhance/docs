---
title: 'Get AI Agent'
api: 'GET /api/ai-agents/{agent_id}'
description: 'Get detailed information about a specific AI agent'
---

<Note>
  **Core API**: Retrieve detailed configuration and status information for a specific AI agent.
</Note>

## Path Parameters

<ParamField path="agent_id" type="string" required>
  Unique identifier of the AI agent
</ParamField>

## Response

<ResponseField name="success" type="boolean">
  Indicates if the request was successful
</ResponseField>

<ResponseField name="data" type="object">
  <Expandable title="Properties">
    <ResponseField name="id" type="string">
      Unique identifier for the AI agent
    </ResponseField>
    <ResponseField name="name" type="string">
      Name of the AI agent
    </ResponseField>
    <ResponseField name="type" type="string">
      Type of AI agent (currently only "VOICE")
    </ResponseField>
    <ResponseField name="status" type="string">
      Current status: "ACTIVE", "INACTIVE", or "TRAINING"
    </ResponseField>
    <ResponseField name="configuration" type="object">
      <Expandable title="Properties">
        <ResponseField name="voice_type" type="string">
          "NATURAL" or "SYNTHETIC"
        </ResponseField>
        <ResponseField name="language" type="string">
          Language code (e.g., "en-US")
        </ResponseField>
        <ResponseField name="capabilities" type="array">
          Array of enabled capabilities
        </ResponseField>
      </Expandable>
    </ResponseField>
    <ResponseField name="created_at" type="string">
      Creation timestamp (ISO 8601)
    </ResponseField>
    <ResponseField name="updated_at" type="string">
      Last update timestamp (ISO 8601)
    </ResponseField>
  </Expandable>
</ResponseField>

## Example Request

```bash
curl --request GET \
  --url https://api.aienhance.net/v1/api/ai-agents/agent_123 \
  --header 'Authorization: Bearer YOUR_ACCESS_TOKEN'
```

## Example Response

```json
{
  "success": true,
  "data": {
    "id": "agent_123",
    "name": "Sales Agent",
    "type": "VOICE",
    "status": "ACTIVE",
    "configuration": {
      "voice_type": "NATURAL",
      "language": "en-US",
      "capabilities": [
        "appointment_setting",
        "lead_qualification",
        "live_transfer"
      ]
    },
    "created_at": "2024-01-20T10:00:00Z",
    "updated_at": "2024-01-20T10:00:00Z"
  }
}
```

## Error Responses

<ResponseField name="Agent Not Found" type="object">
  <Expandable title="Properties">
    <ResponseField name="success" type="boolean">
      Always false for errors
    </ResponseField>
    <ResponseField name="error" type="object">
      <Expandable title="Properties">
        <ResponseField name="code" type="string">
          Error code: "agent_not_found"
        </ResponseField>
        <ResponseField name="message" type="string">
          "AI agent with the specified ID was not found"
        </ResponseField>
      </Expandable>
    </ResponseField>
  </Expandable>
</ResponseField>

{

  }
